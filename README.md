## Overview
This project analyzes job market trends using Python, SQL, and an interactive Streamlit dashboard.

## Features
- Data cleaning and preprocessing
- SQL-based job market queries
- Visualization of job titles, companies, and salaries
- Interactive dashboard built with Streamlit

## Tech Stack
- Python
- Pandas
- SQLite
- Streamlit
- Matplotlib / Plotly

## Project Structure
analysis/      -> data processing and analysis scripts
charts/        -> visualization utilities
dashboard/     -> Streamlit dashboard
data/          -> cleaned datasets
database/      -> SQLite database
